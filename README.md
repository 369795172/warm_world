# 🌟 价值之旅 · 小小世界 (Little World Journey)

一个专为2-4岁婴幼儿设计的温柔虚拟世界应用，通过游戏化的方式培养宝宝的价值观和社交能力。

## 🚀 在线体验

**🔗 立即体验**: [https://little-world-journey.vercel.app](https://little-world-journey.vercel.app)

## 📱 项目特色

### 🎯 核心功能
- **🏠 三个互动场景**：小镇、温馨小家、美丽花园
- **📊 家长模式**：成长数据分析、个性化育儿建议
- **⏰ 智能计时**：可拖拽计时框，温柔的时间管理
- **🎵 多感官体验**：视觉、听觉、触觉全方位互动
- **📱 响应式设计**：完美适配手机、平板、电脑

### 🎨 交互亮点
- **自由拖拽**：计时框可自由拖动，自动保存位置
- **触摸优化**：300ms长按激活，防止误触
- **自动吸附**：拖动结束后智能吸附到合理位置
- **无障碍支持**：符合WCAG标准，支持辅助技术

## 🛠️ 技术栈

### 前端技术
- **React 18** + **TypeScript** - 现代化开发体验
- **Vite** - 超快构建工具
- **Tailwind CSS** - 原子化CSS框架
- **Zustand** - 轻量级状态管理
- **Lucide React** - 精美图标库

### 部署平台
- **Vercel** - 全球CDN部署，毫秒级响应

### 开发工具
- **ESLint** - 代码质量检查
- **TypeScript** - 类型安全保证

## 🎮 场景介绍

### 🏘️ 小镇主页
- 中央广场，宝宝可以选择想去的地方
- 温和的色彩搭配，营造安全感

### 🏠 温馨小家
- 拖拽玩具给小伙伴，学习分享
- 角色互动，培养社交能力
- 即时反馈，强化正向行为

### 🌸 美丽花园
- 种植不同植物，体验生命成长
- 浇水照顾，培养责任感
- 观察变化，激发好奇心

## 👨‍👩‍👧‍👦 家长模式

### 📊 成长快照
- 游戏时长统计
- 兴趣偏好分析

### 💡 共玩建议
- 基于游戏行为的个性化建议
- 现实生活中的亲子活动推荐

### ⚙️ 设置管理

## 🎯 设计理念

### 🌈 温柔真实
- 轻微反馈体现因果，避免恐吓或美化
- 节奏缓慢，音量适中，营造放松氛围

### 🧠 发展适宜
- 专为2-4岁婴幼儿认知特点设计
- 单次交互控制在3步以内
- 强调"被看见"：每个正向行为都有积极反馈

### 👪 亲子共玩
- 为家长提供观察窗口，了解孩子兴趣倾向
- 游戏与现实生活紧密结合
- 促进高质量亲子互动

## 🚀 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/369795172/warm_world.git
cd warm_world

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```

### 部署说明
项目已配置Vercel自动部署，推送代码到main分支即可自动部署。

# 🌟 价值之旅 · 小小世界 (2-4岁版)

一个专为2-4岁婴幼儿设计的温柔虚拟世界应用，基于"价值之旅"终身虚拟世界框架，通过安全、有趣的方式让孩子体验价值创造、分享、照顾等核心概念。

> **价值之旅核心理念**: 以兴趣为内核、用温柔方式呈现现实因果、陪伴人一生做出更好选择的虚拟世界。

## 🎯 项目愿景与哲学基础

### 在价值之旅生态系统中的定位

「小小世界」是价值之旅终身虚拟世界的 **起点** - 专为2-4岁婴幼儿设计的温柔导览，是整个生命旅程的第一站。

### 核心哲学原则

#### 1. 兴趣驱动优先，恐惧驱动仅作边界
- **宝宝体验**: 无强制任务，通过好奇心引导探索
- **家长指导**: 鼓励与安全感比惩罚更重要
- **世界回应**: 每个探索都获得温柔回应，而非评判

#### 2. 时间与复利：长期主义视角
- **成长节奏**: 植物需要时间成长，友谊需要持续互动
- **耐心培养**: 理解"等待"的价值，享受过程而非结果
- **习惯养成**: 从小建立"持续投入=美好结果"的认知模式

#### 3. 世界温柔但不说谎：真实因果呈现
- **分享体验**: 分享带来快乐，但不分享也不会受惩罚
- **照顾植物**: 浇水会促进成长，忽视会停滞但不会死亡
- **情感真实**: 小伙伴会开心、会说谢谢，也会等待和盼望

#### 4. 多元处境与共情理解
- **不同角色**: 小兔兔、小熊熊、爸爸妈妈各有性格
- **处境差异**: 每个角色有不同的需求和反应模式
- **同理心培养**: 通过照顾他人理解"他人的处境"

### 生命五循环的幼儿化表达

| 生命循环 | 成人理解 | 幼儿体验 |
|---------|----------|----------|
| **价值创造** | 可持续收益来自真实价值 | 帮小伙伴捡玩具、给花浇水 |
| **价值传递** | 沟通、承诺、信誉 | 分享玩具、表达感谢 |
| **价值交换** | 公平透明的合作 | 用苹果换气球、互相帮助 |
| **价值循环** | 自律、复利、长期规划 | 种子→浇水→开花→更多种子 |
| **选择与后果** | 短期快感伴随长期代价 | 分享=快乐，不分享=孤单但不惩罚 |

### 社会系统框架的雏形

虽然2-4岁版本简化了复杂的社会系统，但已经埋下了未来扩展的种子：

#### 环境层（Environment Layer）
- **小镇环境**: 安全、资源丰富、规则透明
- **家庭环境**: 温暖、支持、高容错率
- **花园环境**: 自然、需要耐心、有成长周期

#### 评价层（Evaluation Layer）
- **生产性**: 通过帮助行为创造价值
- **信任网络**: 通过分享建立友谊
- **心理健康**: 通过积极互动获得快乐
- **环境适配**: 不同场景有不同的"成功"标准

#### 资本层（Capital Layer）
- **时间资本**: 游戏时长管理、耐心等待
- **社交资本**: 与小伙伴的关系质量
- **经验资本**: 学会的新技能和知识
- **情感资本**: 获得的快乐和满足感

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- pnpm (推荐) 或 npm

### 安装依赖
```bash
# 使用 pnpm (推荐)
pnpm install

# 或使用 npm
npm install
```

### 启动开发服务器
```bash
# 使用 pnpm
pnpm run dev

# 或使用 npm
npm run dev
```

### 构建生产版本
```bash
# 使用 pnpm
pnpm run build

# 或使用 npm
npm run build
```

访问 `http://localhost:5173/` 开始使用应用。

## 📖 使用指南

### 初次使用
1. 打开应用后会进入欢迎页面
2. 设置宝宝的名字和年龄
3. 进入宝宝模式开始探索

### 场景介绍

#### 🏠 小镇主页
- 中央广场是导航中心
- 点击「小家」或「花园」进入不同场景
- 温和的色彩和动画营造安全感

#### 🏡 家场景
- **分享体验**: 拖拽玩具给小伙伴
- **情感反馈**: 小伙伴会开心、说谢谢
- **学习价值**: 分享带来快乐和友谊

#### 🌱 花园场景
- **种植体验**: 选择种子种植在土地里
- **照顾体验**: 用水壶给植物浇水
- **成长观察**: 植物会随时间慢慢长大
- **耐心培养**: 理解成长需要时间和照顾

### 家长模式
长按屏幕或等待游戏时间结束可进入家长模式：

1. **成长快照**: 查看宝宝的游戏行为和兴趣偏好
2. **共玩建议**: 获得基于虚拟体验的现实活动推荐
3. **设置管理**: 调整游戏时间、声音等设置

## 🎮 交互设计原则

### 适合婴幼儿的设计
- **简单操作**: 拖拽、点击等基础手势
- **视觉反馈**: 颜色变化、动画效果
- **语音引导**: 温柔的中文语音提示
- **无文字依赖**: 主要靠图像和声音传达

### 价值理念体现
- **兴趣驱动**: 无强制任务，自由探索
- **温柔真实**: 轻微因果反馈，无惩罚机制
- **积极强化**: 每个正向行为都有鼓励反馈
- **家长共玩**: 鼓励家长参与和引导

## 📊 数据与隐私

### 数据收集
- 游戏行为数据（分享、帮助、照顾等）
- 兴趣偏好分析
- 游戏时长统计

### 隐私保护
- **本地存储**: 所有数据保存在设备本地
- **无网络传输**: 不收集或上传个人信息
- **家长控制**: 可随时清除数据

- 增加新的语音提示

# 价值之旅 · 小小世界（2-4岁版）产品文档

## 产品概述

「价值之旅 · 小小世界」是一个专为2-4岁婴幼儿设计的虚拟世界应用，基于「价值之旅」世界观框架，通过温柔、安全的方式让孩子体验价值创造、分享、照顾等核心概念。

## 核心价值

1. **兴趣驱动**：通过感官探索和简单互动激发好奇心
2. **温柔真实**：轻微反馈体现因果，避免恐吓或美化
3. **生命五循环**：价值创造、传递、交换、循环、选择的婴幼儿化表达
4. **家长共玩**：为家长提供观察孩子兴趣倾向的窗口

## 目标用户

- **核心用户**：2-4岁婴幼儿
- **决策者**：家长/照护者（25-40岁）

## MVP功能范围

### 宝宝端功能
- 小镇主页（中央广场、家、花园）
- 家场景：分享玩具、互动角色
- 花园场景：种植、浇水、观察成长
- 简单拖拽、点击交互
- 温柔语音引导和表情反馈

### 家长端功能
- 成长快照：游戏时长、偏好领域、分享行为统计
- 共玩建议：基于虚拟世界偏好的现实活动建议
- 基础设置：时长控制、静音模式

### 系统功能
- 行为采集与兴趣标签引擎
- 时间控制和温柔退出机制
- 配置文件管理

## 技术架构

- **前端**：React + TypeScript + Tailwind CSS
- **状态管理**：Zustand
- **拖拽交互**：原生HTML5拖拽API
- **音频**：Web Audio API
- **数据存储**：本地存储（IndexedDB）

## 交互原则

1. 单次交互控制在3步以内
2. 所有负向反馈轻柔不过度
3. 强调「被看见」：每个正向行为都有积极反馈
4. 节奏缓慢，音量适中，营造放松安全氛围

## 成功指标

- **行为指标**：7日留存、单次使用时长3-8分钟
- **兴趣指标**：3-5次游玩后能识别明显兴趣偏好
- **家长感知**：世界观温柔可靠，帮助了解孩子喜好

## 版本规划

- **v0.1**：小镇+家+花园场景，基础兴趣统计
- **v0.2**：增加小广场场景，优化兴趣算法
- **v0.3**：支持年龄层切换，为未来价值循环做准备

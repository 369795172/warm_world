「价值之旅」终身虚拟世界 · 世界观与产品框架 v2.0

⸻

1. 项目概述

目标一句话：

打造一个以兴趣为内核、用温柔方式呈现现实因果、陪伴人一生做出更好选择的虚拟世界。

面向人群（按人生阶段）：
	•	婴幼儿：2–4 岁
	•	低龄儿童：5–8 岁
	•	高龄儿童 / 初中：9–14 岁
	•	高中：15–18 岁
	•	大学：18–23 岁
	•	成年人：23+ 岁

核心理念：
	1.	兴趣驱动优先，恐惧驱动只作为边界提示。
	2.	长期主义视角：时间是朋友，只要持续积累就有翻盘可能。
	3.	世界温柔但不说谎：真实呈现选择与后果，避免恐吓或美化。
	4.	多元处境与共情：不同出身、资源、家庭与性格，会导致完全不同的人生路径，本世界的目标之一，是让玩家更真实地体验“他人的处境”，提升同理心。

北极星不变：

衡量成功的最终指标，是玩家在现实中的「选择质量」是否稳步提升。

2. 世界观与哲学基底

2.1 三大世界观原则
	1.	兴趣为主驱，恐惧为边界
	•	游戏鼓励好奇、探索、自主节奏。
	•	负向反馈存在，但不以恐惧/羞辱为主要机制，而是温和的「现实提醒」。
	2.	时间与复利：长期主义
	•	不同年龄段体验不同深度，但世界一直在传递同一信息：
人生是长跑，短期失败不会终结你，持续的积累终将回报你。
	3.	真实因果，而非说教
	•	不回避灰产、犯罪、浪费人生等议题，但不提供具体“操作指南”。
	•	通过世界状态变化、关系网络变化等间接方式展示后果。

2.2 「生命五循环」作为世界底层逻辑
	1.	价值创造 Value Creation
	•	儿童：
我做的事对别人有帮助。
	•	成人：
所有可持续的收益都来自真实价值，而非投机。
	2.	价值传递 Value Transmission
	•	儿童：
告诉别人我能帮忙，让别人知道我在做什么。
	•	成人：
沟通、承诺、信誉，是价值被看见和被信任的关键。
	3.	价值交换 Value Exchange
	•	儿童：
以物换物、互相帮忙，不一定是钱。
	•	成人：
公平、透明、长期合作，优于一次性博弈。
	4.	价值循环 Value Growth
	•	儿童：
种子 → 水果 → 更多种子。
	•	成人：
自律、复利、长期规划、稳健，是人生资产增长的核心。
	5.	选择与后果 Choice & Consequence
	•	儿童：
做坏事可能立刻得到东西，但之后会有不好的结果。
	•	成人：
短期快感/快钱往往伴随长期代价（自由、健康、关系、信誉）。

整个世界的一切系统与剧情，都应围绕这五个循环展开。

2.3 人生发展框架：尝试 → 自知 → 边界 → 深耕

在宏观上，本虚拟世界假设一个「可被玩家体验和反思」的人生发展路径：
	1.	年轻阶段：大量尝试，降低试错恐惧
	•	趁时间充裕、社会成本相对较低的阶段，多去尝试不同的领域、角色与生活方式。
	•	通过「做事」而非「空想」来发现兴趣与厌恶点。
	2.	通过选择与行为认识自己
	•	玩家不是在一开始就被“问你是谁、你喜欢什么”，而是在一次次具体选择中逐渐看清：
	•	我真正愿意投入时间的是什么？
	•	我在压力下会怎么选？
	•	我愿意为哪些结果承担后果？
	3.	设立边界：兴趣与恐惧的平衡
	•	兴趣的好处：主动、积极、自驱，是持续投入与专业化的燃料。
	•	恐惧的好处：提醒风险、保护安全，是边界感与风险意识的来源。
	•	世界会不断展示：
	•	过度追逐目的、过于激进，可能触碰社会底线，引发不可逆的后果（如入狱、严重失信）；
	•	过度恐惧，则会错失成长与创造价值的机会。
	4.	普通人的长期路径：专业化与资本化
	•	对于世界上 90% 以上的普通人，如果以资产或财富自由为目标，现实可行路径往往是：
	1.	找到自己愿意长期投入的专业方向；
	2.	明确目标人群与要提供的价值/产品；
	3.	通过专业能力积累「原始资本」（经验、人脉、资金、信誉）；
	4.	在此基础上引入组织与资本杠杆（雇佣他人、系统化流程），提升产出效率。
	•	游戏中不会简单灌输“成功模板”，而是通过不同背景角色，展示多种普通人的专业化路径与生活选择。
	5.	人生不只有财富目标，也不只有二元对立
	•	世界会有意识地避免“成功/失败”“赢/输”这类简单二元框架，更多呈现：
	•	不同人的人生目标组合（家庭、创作、服务社会、自我修行等）；
	•	同一选择在不同背景下的复杂动机；
	•	「每个人都有自己的考虑」，而不是单一评判标准。

2.4 社会环境如何塑造行为与认知

在本虚拟世界里，我们默认一个前提：

> “人不是在真空里做选择的，**环境先塑造了认知，再通过认知影响行为**。”

为此，世界会抽象出若干关键“环境维度”，贯穿不同年龄、城市、家庭与职业：

1. **资源密度（Resource Density）**
   - 教育、医疗、信息、工作机会等资源的丰富程度。
   - 高资源环境：信息丰富，但竞争激烈、焦虑感更强。
   - 低资源环境：信息闭塞，试错成本高、但节奏慢、关系紧密。

2. **规则透明度（Rule Transparency）**
   - 规则是否清晰、一致、可预期。
   - 高透明环境：更容易靠努力、专业与合作攀升。
   - 低透明环境：更多靠关系、运气、灰色操作，容易产生犬儒与投机心态。

3. **社会价值取向（Value Orientation）**
   - 这个环境默认“什么是成功”。
   - 有的地方崇尚“稳定与服从”，有的地方强调“冒险与创新”。
   - 同一玩家在不同环境，会被引导出完全不同的选择习惯。

4. **安全感与容错率（Psychological Safety）**
   - 犯错的代价高低、失败之后是否还有机会。
   - 高容错环境：鼓励尝试，利于兴趣探索与长期主义。
   - 低容错环境：大家倾向走“最安全”路径，短期主义、保守与内耗更明显。

设计目标：

- 让玩家通过**同一个角色在不同环境中的体验**，看到：
  - 自己的很多想法和选择，不是“性格决定”，而是“环境和信息结构决定”；
  - 换环境 = 换一套默认评价和诱导机制。

2.5 资本目标与认知高度：预测未来的能力

对于那些**以“积累资本”为主要人生目标**的玩家，本世界给出一个清晰的底层定义：

> 资本的“天花板”，由一个人**能否看见未来，并为之提前行动**所决定。

这里的“认知”，不是虚无缥缈的“聪明”“悟性”，而是一条可拆解的链：

1. **看懂当下（理解当前领域状态）**
   - 这个领域现在在什么阶段：野蛮生长 / 红利期 / 过剩竞争 / 夕阳期？
   - 现行规则是什么：主要靠技术、关系、品牌，还是政策红利？
   - 谁在赚真正的钱：是做产品的人、做渠道的人、还是做投机和信息差的人？

2. **推演未来（对趋势做出假设）**
   - 如果技术继续进步 / 人口结构变化 / 政策收紧，会发生什么？
   - 哪些角色会消失，哪些新角色会出现？
   - 这个假设不是“算命”，而是结合信息与常识做出的**可检验预测**。

3. **提前布局（在未来来临前先站到正确的轨道上）**
   - 可以是：
     - 提前积累某项技能或作品；
     - 提前建立某种关系与信任网络；
     - 提前做出一个小产品 / 小服务，针对未来会出现的需求。
   - 也可以是：
     - 在别人还没看懂之前，**用信息差做中间人**，连接需求和供给，但前提是双方都得到真实价值。

4. **未来兑现时获得超额回报**
   - 当世界线按照玩家预期的那种方向发展时：
     - 已经在正确轨道上的人，天然享受“顺势而上”的红利；
     - 晚到的人，即便做同样的事，也只能拿到普通回报。

游戏中对“资本”的态度：

- 不鼓励“空转的信息差套利”，即只割别人韭菜、不创造实质价值。
- 鼓励的是：**用更高的认知去提前识别价值缺口，并用产品/服务填上这个缺口**。
- 长期来看，世界会通过**信用、关系、机会开放度**等维度，奖励后者，削弱前者的可持续性。

3. 底层社会系统框架（V1 草图）

> 本章节是“世界的社会物理定律”，所有剧情与玩法都应在这套框架内自洽。

我们将社会系统抽象为四个层次：

1. **环境层：人在什么样的“水”里游泳**
2. **社会评价层：这摊水里用什么“标尺”评判人**
3. **资源与资本层：资源如何流动、沉淀与放大**
4. **认知与未来预测层：谁能更早看到水流向哪里**

### 3.1 环境层：不同“池子”里的默认设置

关键对象：城市 / 学校 / 公司 / 圈层 / 家庭。

每个环境都有一组核心参数（示例）：

- 资源密度：教育、机会、资本的集中程度。
- 规则透明度：制度清晰度与执行一致性。
- 社会价值取向：更看重“稳定”“成绩”“创新”还是“关系”。
- 安全感与容错率：失败的代价与重来机会。
- 信息流通度：外部信息进入的难度，本地信息外流的难度。

**系统效果：**

- 同一行为在不同环境，触发的反馈完全不同：
  - 在高压名校里，考前熬夜 = 得到赞许与同情；
  - 在创新型社区里，长期熬夜可能被 NPC 提醒“你在透支健康，会损害创造力”。
- 玩家通过迁移动线和故事线，逐渐理解：
  - **“我在哪”先于“我是谁”。**

### 3.2 社会评价层：多维度而非单一分数

我们避免一个简单的“社会分数”，而是拆成几个可观察但隐含权重的维度：

1. **生产性与价值贡献（Contribution）**
   - 玩家对他人/社会创造的可持续价值：产品、服务、创意、陪伴、公共贡献等。

2. **信誉与信任网络（Trust & Network）**
   - 信守承诺记录、合作历史、债务偿还、言行一致程度。
   - 决定玩家能否参与高杠杆机会（合伙、融资、关键任务）。

3. **规则合规度（Legality & Compliance）**
   - 是否触碰法律、社会底线、严重违背公共规范。
   - 影响玩家的“自由度”和未来可选路线。

4. **自洽与心理健康（Coherence & Well-being）**
   - 行为是否与自己宣称的价值观相符；
   - 是否长期处在极度压抑或极度空虚状态。
   - 不直接“加钱”，但体现在关系质量、决策质量与事件触发。

5. **环境适配度（Environment Fit）**
   - 这个玩家的行为风格与当前环境价值观是否匹配。
   - 高适配 = 更容易被提拔/信任；
   - 低适配 = 会被边缘化，或被迫“离开/迁移”。

**重要设计：**

- 不对玩家展示一个“总评 A/B/C”，而是通过**NPC态度、机会开放度、对话台词、环境细节变化**来传达评价变化。
- 不同环境对这几项的**权重不同**（例如：某公司把“服从与加班”权重极高，却忽视健康与家庭），玩家能用对比体验到“环境偏见”。

### 3.3 资源与资本层：钱只是其中一种资本

在世界观中，“资本”被拆成多种形态：

1. **经济资本**：金钱、资产、股权、作品版权等。
2. **人力资本**：技能、经验、声誉、作品集。
3. **社会资本**：信任关系、合作网络、组织位置。
4. **时间资本**：年龄、可自由支配时间、健康状况。
5. **认知资本（见 3.4）**：理解世界和预测趋势的能力。

资源流动规则（简化）：

- 环境 + 社会评价 → 决定玩家**有哪些资源入口**（工作、项目、机会）。
- 玩家选择与行为 → 改变自己的**社会评价向量**，进而影响未来的入口。
- 某些路径会造成「短期资源 + 长期损伤」（比如灰产、短期信息差套利）；
- 某些路径会造成「短期吃亏 + 长期放大」（长期学习、做小但真实的产品、维护信誉）。

### 3.4 认知与未来预测层：谁能更早看到“潮水方向”

在这个层里，我们关心的是：**谁先看到未来，谁先站上浪头。**

认知在系统中的具体表现：

1. **信息视野等级（Information Horizon）**
   - 低等级：只能看到身边人的抱怨与鸡毛蒜皮。
   - 中等级：能接触到行业新闻、世界变化、不同城市/国家的生活方式。
   - 高等级：能看到制度变迁、技术趋势、人群结构变化等更慢更大的东西。

2. **推理与假设机制**
   - 游戏不会给“标准答案”，而是通过任务要求玩家：
     - 针对某个领域写下自己的“未来三种可能”；
     - 选择其中一种作为自己的“下注方向”；
     - 用实际行为（职业选择、项目投入、学习路径）来验证。

3. **提前布局与兑现**
   - 系统中存在“世界时间线”：技术、政策、社会观念会缓慢变化。
   - 玩家在时间线的早期选择：
     - 做出符合未来趋势的产品/作品/技能投入 →
       在未来阶段获得更高的资源产出和机会。
     - 选择只利用短期信息差、但不创造真实价值 →
       短期收益可观，但随着规则收紧 / 信息透明度提高，这些路径会逐渐枯竭。

4. **认知与资本的关系**
   - 认知不直接给钱，而是：
     - 决定玩家可以接触到怎样的机会；
     - 决定玩家是否敢于在正确的时间“换道”“转型”；
     - 决定玩家有没有能力识别“短期陷阱 vs 长期红利”。

### 3.5 玩家体验目标：从“努力不够”到“看见结构”

这套社会系统的核心体验目标：

- 让玩家从“我是不是不够努力/不够聪明”，逐渐转向：
  - “原来环境的评价标准有偏见”；
  - “原来我可以主动选择环境，而不是只被环境选”；
  - “原来决定我资本上限的，是我在多大程度上看见并拥抱未来，而不是单次考试分数”。

5. 生命周期分层设计

5.1 年龄阶段 × 产品职责

阶段	现实核心需求	世界要扮演的角色	核心主题
婴幼儿（2–4）	感官探索、安全感	温柔的导览者	世界很大、很有趣，我是安全的
低龄儿童（5–8）	好奇、模仿、简单规则	小镇探险伙伴	价值=帮助别人，合作比独占好
高龄儿童/初中（9–14）	想快点长大、试错	成长沙盒与试验场	选择与后果、责任、兴趣探索
高中（15–18）	高考压力、焦虑	高压环境下的“平衡器”	人生不只有高考，时间+复利+自律
大学（18–23）	价值观定型初期、诱惑多	人生路线模拟器	不做让自己后悔的事（犯罪、成瘾等）
成年人（23+）	低谷、倦怠、再选择	温暖但说真话的精神老家	接纳、重启、重新规划

同一世界观，不同阶段以不同深度和交互形式呈现，但哲学内核一致。

5.2 面向家长的隐性目标
	•	传递：兴趣是最好的老师，鼓励与安全感比惩罚更重要。
	•	展示：孩子在世界中的行为记录，给家长一个观察其动机和兴趣倾向的窗口。
	•	引导：家长在现实中减少恐惧驱动（威胁、羞辱、高压），增加兴趣驱动（陪伴、共情、共同探索）。

6. 动机系统：兴趣驱动 vs 恐惧驱动

在本世界观中，「兴趣」与「恐惧」都被视为重要的心理力量：
	•	兴趣带来主动与创造。
	•	恐惧带来边界与安全感。

目标不是消灭恐惧，而是让玩家：
	•	识别哪些是现实风险提醒，哪些是过度内化的惩罚性声音；
	•	学会在兴趣驱动与安全边界之间找到平衡。

6.1 内在“声音系统”：Interest Voice vs Fear Voice

系统抽象出两类内在声音：
	1.	鼓励型声音（Interest Voice）
	•	对应现实中来自家人、老师、伙伴的正向反馈，也可能是玩家逐渐形成的内在支持性自我对话。
	•	示例台词：
	•	「你可以试试看，哪怕慢一点也没关系。」
	•	「你刚才做得不错，结果一般没关系，我看到你的努力了。」
	2.	惩罚型/恐惧型声音（Fear Voice）
	•	对应现实中来自过度惩罚式养育、羞辱、威胁等形成的消极自我对话。
	•	示例台词：
	•	「如果你这次做不好就完蛋了。」
	•	「这么简单你都做不好，将来怎么办？」

设计目标：
	•	帮玩家识别：
	•	哪些话是在提醒真实风险；
	•	哪些话只是放大的焦虑或他人的投射。
	•	通过智慧 NPC 与剧情，引导玩家逐步学会：
	•	用更温柔、现实的语言替换惩罚性声音；
	•	在关键选择时，从「我要躲开惩罚」切换为「我想靠近我在乎的东西」。

4.2 兴趣种子系统

核心概念：兴趣是一颗种子，持续关注与投入会让世界打开新路径。

机制示意：
	1.	玩家对某个行为/领域表现出持续好奇和参与。
	2.	系统记录该方向的行为时长与频率。
	3.	世界在这个领域逐步解锁：
	•	新 NPC、新地点、新任务线。
	•	更深层的故事或能力，而不是简单的数值奖励。

区别于传统“刷数值”设计：
	•	不用「多做就多金币」的直接反馈。
	•	更强调「我越投入，世界越承认我的身份」的感受。

7. 高风险 / 灰产主题的处理原则 / 灰产主题的处理原则

7.1 三个底线
	1.	不提供具体操作指南
不教“如何做灰产/违法行为”，所有相关描述保持高度抽象。
	2.	不浪漫化、不英雄化
不把“非法致富”“铤而走险”包装成酷、帅、刺激；拒绝反英雄浪漫化。
	3.	不猎奇、不夸张
不用恐怖场景和极端惩罚吓人，而是通过真实、缓慢的后果呈现问题。

5.2 因果链呈现方式

以「短期快钱诱惑线」为例：
	•	起点：看似友好的 NPC 提供「轻松赚快钱」的机会。
	•	玩家选择路径：
	1.	直接拒绝。
	2.	试探了解。
	3.	直接参与。
	•	若选择 2 / 3：
	•	短期：
	•	资源确实增加，生活条件改善。
	•	中期：
	•	与某些正向 NPC 的关系变差/疏远。
	•	部分公共空间或机会对玩家“关门”。
	•	长期：
	•	出现被威胁、被利用、被调查的剧情线。
	•	玩家逐步感到自己“被系统绑架”，自由减少。
	•	某些极端路径可能导致入狱情节：
	•	游戏以「时间被从真实社会中抽离」的形式呈现——世界在向前发展，但玩家在封闭空间中消耗时间；
	•	玩家在世界中的「信用评价」遭到严重打击：
	•	很多需要信用的机会（职业、角色、重要任务线）不可用或需要极长时间修复；
	•	一部分关系永远无法完全恢复。

智慧老者 NPC 的角色：
	•	不直接训斥，不说「你错了」。
	•	更偏【镜子型提问】：
	•	「你觉得现在的你更自由了吗？」
	•	「你花的这些钱，像是你在投资未来，还是在透支未来？」
	•	「如果把时间看成你最宝贵的资产，你愿意用多少时间去偿还这一次的决定？」

目标：
	•	让玩家自发得出结论：
原来所谓“灰产/捷径”，很多时候是拿人生去换短期快感。
	•	保留“重新选择”的机会（重启线），但代价可感知，让玩家体会「改正永远值得，但也不是没有成本」。

8. 智慧 NPC 系统设计

智慧 NPC 不只是“说教的嘴”，而是一个“三层结构”的哲学系统：

8.1 故事线（表层）
	•	NPC 有自己的生活史与小故事：年轻时的选错路、错过的机会、坚守的选择等等。
	•	玩家通过与 NPC 相处和任务推进，逐步解锁这些故事片段。

8.2 因果线（系统中层）
	•	NPC 会根据玩家的行为模式给出对应反馈。
	•	示例：如果玩家连续多次选择“捷径”获得资源：
	•	NPC 不会直接否定，而是以「共情+自我代入」的方式说：
「我年轻时也这样……后来发现，有些捷径会让人丢掉更重要的东西。」
	•	NPC 既是叙事角色，也是「系统的表达接口」。

8.3 镜像线（哲学内核）
	•	NPC 不给标准答案，而是抛出问题让玩家照见自己：
	•	「如果 10 年后的你看到现在的选择，会怎么评价？」
	•	「你做这个决定时，是在追求什么？是在逃避什么？」
	•	目标是慢慢塑造玩家的自我反思能力。

6.4 NPC 的「言行不一」与内在冲突

真实世界中，大量人物是「言行不完全一致」的：
	•	口头上强调某种价值，行为上却因为恐惧、压力、利益而做出相反选择；
	•	对他人说的“道理”很正确，对自己却做不到。

在游戏设计中：
	•	我们不会把 NPC 设计成单一标签（好人/坏人/导师/反派），而是：
	•	允许他们在不同情境下展示出矛盾的选择；
	•	通过支线或回忆，展示其成长背景、认知局限、现实压力。
	•	对玩家而言，这是一个**观察与理解“言行不一背后的故事”**的机会：
	•	明白很多不一致，源于「对自己不够了解」：
	•	不清楚自己真正想要什么；
	•	不清楚自己真实能力与限制；
	•	不清楚达成目标的合理路径与后果。
	•	玩家可以从 NPC 身上看到「如果不花时间了解自己，会发展出怎样的内在撕裂」。

设计目标：
	•	帮助玩家在体验他人处境时提升同理心，而非简单评判；
	•	反向促使玩家思考：
「我说的和做的之间，有没有类似的落差？它从哪里来？」

6.5 不同年龄段的语言风格示例
	•	对婴幼儿 / 低龄儿童：
	•	「我们去看看世界上不同的小伙伴，看他们都在做什么。」
	•	「你愿意把这块点心分给他吗？看，他笑了。」
	•	对初中生：
	•	「你想快点长大，对吗？在这里，你可以先试一试‘大人的选择’。」
	•	对高中生：
	•	「高考确实重要，但它不是你这一生唯一的门。」
	•	「你现在走的每一步，都是在给未来的自己铺路，而不是在决定‘成败’。」
	•	对大学生：
	•	「你正在站在价值观的十字路口。有些路故事很多，但代价是自由。」
	•	对成年人：
	•	「你不是第一次跌倒。你能试着站起来这一件事本身，就已经证明你还在前进。」
	•	「我们可以一起重新规划一个‘长线版本’的人生路径。」

9. 产品层成功定义与度量

9.1 北极星：选择质量

选择质量由以下维度构成：
	•	更长期主义：更常考虑长期后果，少做一次性博弈。
	•	更利他：更常考虑他人影响与合作机会。
	•	更稳健：减少高风险、不可逆的冲动行为。
	•	更能承受短期痛苦：愿意接受延迟满足。
	•	更自洽：对自己选择有理解、有担当，而非完全外归因。

7.2 行为指标（游戏内）
	•	延迟满足相关行为：
	•	完成长周期任务 vs 放弃率。
	•	为长期目标储蓄/投入资源的频率。
	•	合作与利他行为：
	•	主动帮助 NPC / 其他玩家的比例。
	•	资源分享 vs 独占的行为数据。
	•	高风险选择：
	•	面对捷径/灰色选项时，不同路径的选择比例。
	•	随游戏时长增长，“高风险选择”比例是否下降。

7.3 心理与态度指标（问卷/访谈）

可定期或阶段性投放轻量问卷（针对不同年龄和家长）：
	•	「我觉得人生有很多种路可以走」的认同度。
	•	「我愿意为长期目标承受一些短期不舒服」的认同度。
	•	「我做决定时会考虑这会不会伤害别人」的认同度。

7.4 家长与成人反馈
	•	家长观察：
	•	孩子是否更愿意沟通“自己喜欢什么”，而不是只问“这样对不对”。
	•	孩子是否在现实中表现出更多合作/分享/独立思考。
	•	成人玩家：
	•	是否会在低谷期“回到这个世界走走”。
	•	是否有具体时刻，将游戏中的一句话、一个剧情应用到现实决策。

### 9.4 MVP 在底层框架中的切片定位

> ——这部分说明：我们现在只做一小块，但思想上保持完整。

为了控制复杂度，MVP 依然：

- 只覆盖 **一个年龄段**（如：高中或大学）；
- 只实现 **一条相对单线的主线剧情**（例如“灰产/捷径诱惑线”或“高考焦虑线 + 兴趣选择”）。

但在设计时，MVP 明确被视为整个社会系统的一个**切片**：

1. **环境层**
   - 固定为一个具体城市 + 学校/社区的组合，参数写死（资源密度、规则透明度、高考压力等）。
   - 只做 1~2 个环境对比（例如：重点中学 vs 普通中学 / 大城市 vs 小城），让玩家略微感到“环境差异”。

2. **社会评价层**
   - 选出 2~3 个最相关维度（例如：成绩、服从度、心理健康），通过 NPC 台词与机会开放度体现。
   - 不做复杂的显式评分，只用少量反馈体现“老师/家长/社会的偏见”。

3. **资源与资本层**
   - 暂时只做“时间资本 + 学习成绩 + 人际关系”三种资源；
   - 不引入复杂的金钱或投资系统，但通过故事预埋“未来职业/专业选择”的资本差异。

4. **认知与未来预测层**
   - 用非常轻量的形式出现：
     - 例如给玩家不同的“未来人生片段”；
     - 要求玩家在当前做选择时，考虑不同未来的代价。
   - 不做完整的“世界时间线系统”，而是通过数个关键跳转节点，展示“提前布局 vs 临时抱佛脚”的差异。

5. **对未来扩展的约束与对齐**
   - 所有新年龄段、新城市、新职业线，未来都要：
     - 用同一套**环境参数**语言描述；
     - 用同一套**社会评价维度**评估玩家；
     - 在同一条“认知—预测未来—提前布局—资本兑现”的逻辑上自洽。

10. 版本策略与 MVP 建议

10.1 不建议一开始同时覆盖所有年龄段

复杂度极高，难以验证价值。更优路线：
	1.	先选一个“议题最尖锐 + 反馈能力最强”的阶段切入。
	2.	在这个阶段验证：
	•	世界观表达是否清晰。
	•	玩家是否真的被触动 + 行为有改变。
	3.	再向下（儿童）和向上（成年）扩展界面与玩法。

10.2 初始切入建议：高中 + 大学阶段

理由：
	•	文本叙事 +轻交互就能构成完整体验，对美术和复杂系统依赖低。
	•	这一段是价值观与人生路线的关键期，议题天然强烈：
	•	高考/考试焦虑。
	•	「人生只有一条正确道路吗？」
	•	灰产/投机/成瘾的诱惑。
	•	他们有能力表达自己的感受与反馈，便于快速迭代。

10.3 三条候选主线（供后续选择 MVP 切入口）
	1.	灰产/捷径诱惑线
	•	核心问题：
「快速得到的钱/结果，值不值得用长期人生去换？」
	2.	高考/考试焦虑线
	•	核心问题：
「如果考试不是人生唯一评判标准，我要如何规划自己？」
	3.	兴趣探索与长期规划线
	•	核心问题：
「我到底喜欢什么？兴趣值得投入多久？会不会耽误‘正事’？」

MVP 可以从其中一条主线做一个「小镇 + 对话冒险」的轻量版本，用于：
	•	验证世界观是否打动目标人群。
	•	收集真实选择路径与反馈，为后续多年龄层扩展提供经验基础。
